* Обзор проекта
Данный репозиторий представляет собой рабочее пространство, развернутое в DevContainer. Оно объединяет инструменты разработки, документацию и набор серверов [[https://modelcontextprotocol.io/][Model Context Protocol (MCP)]] для интеграции с внешними сервисами (Jira, GitLab, Graylog, Mattermost).

* Быстрый старт
Проект предназначен для запуска в Docker через DevContainer.

** Сборка и запуск
1. Скопируйте шаблон окружения:
   #+BEGIN_SRC bash
   cp .devcontainer/.env-dist .devcontainer/.env
   #+END_SRC
2. Заполните необходимые секреты в =.devcontainer/.env=.
3. Добавьте bash_history:
   #+begin_src bash
make
   #+end_src
4. Выполните сборку и запуск:
   #+BEGIN_SRC bash
   .devcontainer/activate
   #+END_SRC
5. Внутри контейнера:
   #+begin_src bash
opencode web --port=9631 --hostname=0.0.0.0
#+end_src

** Как использовать
1. Для доступа к Opencode открыть в браузере http://localhost:9631
2. Для доступа к bash: =.devcontainer/exec bash=
3. В bash внутри devcontainer работает ИИ: =bai <command>=
   
** Использование MCP инструментов
В проекте настроены следующие MCP-подключения (см. =.opencode/opencode.json=):
- *Atlassian* (Jira/Confluence)
- *GitLab* (инспекция репозиториев)
- *Graylog* (анализ логов)
- *Mattermost* (уведомления и мониторинг)
- *Pandoc* (конвертация документов)
- *Docling* (парсинг документов через удаленный сервер)
  
И т.д.

* Google Sheets (Google Cloud Platform)
1. Нужно создать service account, сделать ему key в формате json.
2. Протестировать внутри контейнера:
 #+begin_src bash 
CREDENTIALS_CONFIG=$(base64 -w0 ./credentials.json) uvx mcp-google-sheets@latest
#+end_src
3. Вписать переменную в .env
4. Пригласить по почте сервисный аккаунт: дать доступ к файлу или папке. Почта для mcp:
  sheets-mcp@crypto-gradient-487614-q5.iam.gserviceaccount.com
